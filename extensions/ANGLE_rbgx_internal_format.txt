Dependencies

    OpenGL ES 2.0 is required.

Overview

    This extension introduces a sized internal RGBX format. This is a 32-bit representation of the
    24-bit RGB format, with the Alpha channel ignored.

    None.

    None.

New Tokens

    // 32-bit RGBX format
    GL_RGBX8_ANGLE                    0x96BA
